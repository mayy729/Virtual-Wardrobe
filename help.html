2. Uploading Clothes

üì∑ Upload Your First Item

  1. Go to "üîÆ Upload Clothes" from the navigation menu
  2. Click or drag and drop images into the upload area
  3. Supported formats: JPG, PNG, GIF, WebP (max 5MB per file)
  4. After uploading, you'll see a preview of your images

üìù Fill in Item Details

  1. Type: Select whether this item is "Clothes" or "Accessories"
  2. Name: Give your item a descriptive name (e.g., "Pink Summer Dress")
  3. Season: Select one or more seasons by checking the boxes (Spring, Summer, Autumn, Winter, or All Seasons). You can select multiple seasons!
  4. Occasion: Select one or more occasions by checking the boxes (Casual, Date, Work, Party, Formal, Sport, or All Occasions). You can select multiple occasions!
  5. Brand: Enter the brand name (optional)
  6. Size: Enter the size (optional, e.g., "M" or "38")
  7. Material: Enter the material (optional, e.g., "Cotton, Silk")
  8. Notes: Add any additional notes (optional)

üí° Tip: Use checkboxes to select multiple seasons and occasions. For example, a dress can be suitable for both Spring and Summer, and for both Date and Party occasions.

ü™Ñ Auto Cutout Feature

The Auto Cutout feature automatically removes the background from your clothing images:

  1. After uploading, click "üé® Auto Cutout" to process all images
  2. Or click "ü™Ñ Remove Background" on individual preview items
  3. The system will process the image and remove light backgrounds
  4. You can still save items without using cutout if you prefer

üì∏ Adding Wearing Photos

  1. Check the "Add wearing photo" checkbox
  2. Select a photo of you wearing the item
  3. This photo will be displayed alongside the item in your wardrobe
  4. You can also add or change the wearing photo later when editing the item

üì± Uploading on Different Devices

The upload feature works on all devices including Apple devices:

  • Desktop/Windows: Click or drag and drop images
  • Mobile/Android: Tap the upload area to select photos
  • Apple Devices (iPhone/iPad): Tap the upload area to select photos from your library or take new photos
  • Supported formats: JPG, PNG, GIF, WebP, HEIC, HEIF
  • Maximum file size: 5MB per image

3. Managing Your Wardrobe

üëò Viewing Your Wardrobe

  1. Go to "üëò My Wardrobe" from the navigation menu
  2. All your uploaded items will be displayed in a grid
  3. Use pagination to navigate through multiple pages

üîç Searching and Filtering

You can filter your wardrobe in several ways:

  1. Type Filter: Filter by "Clothes" or "Accessories"
  2. Search Bar: Type keywords to search by name, brand, or notes
  3. Season Filter: Use the dropdown to select one or more seasons (Spring, Summer, Autumn, Winter, All Seasons).
    • Selecting "All Seasons" will show all items regardless of their season
    • Selecting specific seasons (e.g., "Winter") will show items that have that season OR have "All Seasons"
    • You can select multiple seasons to see items matching any of them
  4. Occasion Filter: Use the dropdown to select one or more occasions (Casual, Date, Work, Party, Formal, Sport, All Occasions).
    • Selecting "All Occasions" will show all items regardless of their occasion
    • Selecting specific occasions (e.g., "Casual") will show items that have that occasion OR have "All Occasions"
    • You can select multiple occasions to see items matching any of them
  5. Brand Filter: Type to filter by brand name (partial match supported)
  6. Size Filter: Type to filter by size (partial match supported)
  7. Material Filter: Type to filter by material (partial match supported)
  8. Click "Clear Filters" to reset all filters

üí° Tip: Filters work together - items must match ALL active filters to be displayed.

üëÅÔ∏è Viewing Item Details

‚úèÔ∏è Editing Items

üí° Tip: This is especially useful if you made a mistake during upload or want to update information later!

üóëÔ∏è Deleting Items

4. Creating Outfits

üí´ Create a New Outfit

  1. Go to "üí´ Create Outfit" from the navigation menu

  6. Click "Save" to save the outfit
  7. The outfit will appear in your "Saved Outfits" page

üí° Tip: You can select multiple seasons and occasions for one outfit. For example, an outfit can be suitable for both Spring and Summer, and for both Date and Party occasions.

7. Understanding Filters

üéØ How Filters Work

Understanding filter logic will help you find items more effectively:

Season Filter Logic:
  • Selecting "All Seasons": Shows ALL items, regardless of their season
  • Selecting specific seasons (e.g., "Winter"): Shows items that:
    • Have "Winter" in their season list, OR
    • Have "All Seasons" in their season list
  • Selecting multiple seasons (e.g., "Spring" and "Summer"): Shows items that match ANY of the selected seasons OR have "All Seasons"
Occasion Filter Logic:
  • Selecting "All Occasions": Shows ALL items, regardless of their occasion
  • Selecting specific occasions (e.g., "Casual"): Shows items that:
    • Have "Casual" in their occasion list, OR
    • Have "All Occasions" in their occasion list
  • Selecting multiple occasions (e.g., "Date" and "Party"): Shows items that match ANY of the selected occasions OR have "All Occasions"
Examples:
  • An item with seasons ["Spring", "Summer"] will show when you filter by "Spring" OR "Summer" OR "All Seasons"
  • An item with seasons ["All Seasons"] will show when you filter by ANY season or "All Seasons"
  • An item with occasions ["Casual", "Date"] will show when you filter by "Casual" OR "Date" OR "All Occasions"

8. Troubleshooting

‚ùì Common Issues

Can't connect to server
  • Check your API Base URL in Settings
  • Make sure the URL is correct and includes https://
  • Test the connection using the "Test Connection" button
  • Ensure your internet connection is working
Images not uploading
  • Check file size (must be under 5MB)
  • Ensure file format is supported (JPG, PNG, GIF, WebP, HEIC, HEIF)
  • On Apple devices: Make sure you're selecting photos from your library or taking new photos
  • If the file picker appears twice, this is a known issue that has been fixed - try refreshing the page
  • Try refreshing the page and uploading again
  • Check your internet connection
  • Make sure you're logged in to your account
Can't log in
  • Make sure you're using the correct username and password
  • Try using "Forgot Password" to reset your password
  • Check if your account still exists (data may have been lost if using file-based storage without MongoDB)
Items not showing up
  • Refresh the page
  • Check if filters are applied (try clearing them)
  • Verify your API connection is working
