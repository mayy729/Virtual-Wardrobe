# 虚拟衣柜系统 - 为C小姐定制

## 项目的简介

这是一个专为C小姐定制的虚拟衣柜管理系统，帮助她轻松管理衣物，快速创建完美搭配，告别选择困难症。系统采用Hello Kitty主题设计，界面美观友好，操作简单直观。

## 功能特性

### ✨ 核心功能

1. **📤 上传衣物**
   - 支持多图片上传
   - 拖拽上传功能
   - 图片预览

2. **🎨 自动抠图**
   - 智能背景移除
   - 让衣物展示更美观

3. **🏷️ 分类管理**
   - 季节分类（春季、夏季、秋季、冬季、四季）
   - 场合分类（日常、约会、工作、聚会、正式、运动）
   - 品牌、尺寸、材质标注
   - 备注功能

4. **🔍 智能筛选**
   - 多维度筛选（至少5种筛选器）
   - 关键词搜索
   - 实时筛选结果

5. **✨ 创建搭配**
   - 从衣柜中选择衣物
   - 拖拽或点击添加
   - 实时预览搭配效果

6. **💾 保存搭配**
   - 保存喜爱的搭配方案
   - 命名和分类搭配
   - 按季节和场合筛选已保存搭配

7. **📷 穿着照片**
   - 可选添加穿着效果照片
   - 方便查看实际穿着效果

### 🎨 设计特色

- Hello Kitty主题风格
- 粉色系配色方案

## 使用方法

### 1. 打开网站

直接在浏览器中打开 `index.html` 文件即可开始使用。

### 2. 上传衣物

1. 点击导航栏中的"上传衣物"
2. 点击上传区域或拖拽图片文件
3. 选择图片后，填写衣物信息：
   - 名称（必填）
   - 季节（必选）
   - 场合（必选）
   - 品牌、尺寸、材质（可选）
   - 备注（可选）
4. 点击"自动抠图"按钮可移除背景（可选）
5. 点击"保存衣物"完成上传

### 3. 查看衣柜

1. 点击"我的衣柜"查看所有衣物
2. 使用搜索框和筛选器查找特定衣物
3. 点击"查看详情"查看完整信息
4. 可以删除不需要的衣物

### 4. 创建搭配

1. 点击"创建搭配"
2. 从左侧的衣柜中选择衣物
3. 可以拖拽或点击衣物添加到搭配区域
4. 点击"保存搭配"按钮
5. 填写搭配信息（名称、季节、场合、备注）
6. 保存成功

### 5. 查看已保存搭配

1. 点击"已保存搭配"
2. 使用搜索和筛选功能查找特定搭配
3. 点击"查看详情"查看完整搭配信息

## 筛选功能说明

系统提供以下筛选器（超过5种）：

1. **季节筛选**：春季、夏季、秋季、冬季、四季
2. **场合筛选**：日常休闲、约会、工作、聚会、正式场合、运动
3. **品牌筛选**：通过文本输入搜索品牌
4. **尺寸筛选**：通过文本输入搜索尺寸
5. **材质筛选**：通过文本输入搜索材质
6. **关键词搜索**：搜索衣物名称、品牌、备注等

## 技术说明

### 技术栈

- HTML5
- CSS3（包含响应式设计）
- JavaScript (原生JS，无框架依赖)
- LocalStorage（本地存储）

### 浏览器兼容性

- Chrome（推荐）
- Firefox
- Safari
- Edge

### 数据存储

所有数据（衣物信息、保存的搭配）都存储在浏览器的 LocalStorage 中，无需服务器支持。

**注意**：清空浏览器缓存可能会丢失数据，建议定期备份重要数据。

## 特色功能详解

### 自动抠图功能

系统使用Canvas API实现基础的背景移除功能：
- 自动识别浅色背景并移除
- 对于复杂背景，可以使用专业的背景移除API（需要配置API密钥）

### 响应式设计

- 桌面端：完整功能展示，多列布局
- 平板端：自适应布局，保持良好的视觉效果
- 手机端：单列布局，优化触摸操作

## 使用建议

1. **定期整理**：建议定期检查衣柜，删除不再需要的衣物信息
2. **详细标注**：上传时尽量填写完整信息，便于后续筛选
3. **创建搭配库**：为不同场合创建多个搭配方案
4. **使用备注**：在备注中记录特殊信息（如"借给朋友"等）

## 常见问题

**Q: 数据会丢失吗？**  
A: 数据存储在浏览器LocalStorage中，清空浏览器缓存会丢失数据。建议定期备份。

**Q: 支持哪些图片格式？**  
A: 支持常见的图片格式（JPG、PNG等）

**Q: 可以批量上传吗？**  
A: 可以，支持一次选择多张图片上传。

**Q: 抠图效果不理想怎么办？**  
A: 可以跳过自动抠图，直接使用原图。对于更好的抠图效果，可以手动使用图片编辑软件处理后再上传。
